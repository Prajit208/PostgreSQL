## SQL concepts
### Primary key-> Unique and not null value in a table
### Foreign key -> Primary key of one table connecting to other table
### Constraints:

#### Unique Constraint-> no duplicate value allowed , Make sure every record is unique
#### NOT NULL constraint-> No null value allowed
